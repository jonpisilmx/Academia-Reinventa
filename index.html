FILOSOFÍA

The REPLICA Method

The Sound-First Approach: Un Manifiesto Pedagógico

I. El Sonido como Cimiento

La adquisición real ocurre cuando el cerebro procesa el sonido antes del significado. Basados en las investigaciones de Bill VanPatten, establecemos que no se puede reproducir lo que no se puede oír.

II. Lexical Approach

Siguiendo a Michael Lewis, enseñamos el idioma como un sistema de unidades prefabricadas (Chunks). Priorizamos herramientas de comunicación inmediata.

III. Del Oído a la Acción

Tres etapas críticas: Descodificación Fonética, Input Comprensible y Negociación de Significado. Aplicamos las técnicas de entrenamiento de músculos faciales de Adrian Underhill.

IV. El Filtro Afectivo

Eliminamos el miedo al error. Basados en la teoría de Stephen Krashen, creamos un ambiente de inmersión total donde el error es solo un ajuste de sintonía fina.

"Nuestra máxima: No se puede reproducir lo que no se puede oír; y no se puede entender lo que no se ha procesado primero a nivel fonético."
